SearchMagic (Beta)

SearchMagic is a search application that scrapes sites based on a small, extensible plugin that defines a search site. Included are plugins to search Google, Google Images, Amazon Books, and Amazon Electronics. If you want more plugins, email me so I know there is a demand for more. There is also a plugin system in place to make the results from all your searches look a certain way. Here is a movie of the app in action. Play around with this alpha and let me know what you think.

Please submit feedback by emailing me please.

PanicSaver

PanicSaver is an original Positron idea. There have been other screensavers that display crashes. BORING. But this one is different, it emulates a 10.3 kernel panic. Well, I guess it is pretty much the same. Crap. Anyway, it is our first attempt at a cocoa app so it is a beta for now. It probably has some memory leaks, heck it probably will burn your computer. But panic will ensue. And that is the important thing.
Note: Source Code on source page. Only tested on XCode. Submit feedback by emailing me please.

Name That iTune

Name That iTune is a copy of the game "Music Quiz" on the iPod. Basically, it plays a random song out of your library and it puts in random song titles. The quicker you guess the song, they more points you get. It uses applescripts to communicate to iTunes to tell it what to do. If you have an icon idea, feel free to email it to me

Freq.

This program is a helpful utility for counting the number of each letter in a document. This is used to analyze encrypted documents using a substitution cipher. Very simplistic. This was kind of our first "big" program. It has a 5 star rating on VersionTracker!
Note: The Mac versions are 1.1 and the Windows version is 1.0
